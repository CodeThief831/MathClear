# Contributing to MathClear

Contributions that improve mathematical accuracy, accessibility, mobile usability or VTU 2021 Scheme coverage are welcome.

## Before opening a pull request

1. Create a focused branch.
2. Keep mathematical notation compatible with MathJax.
3. Cite the official syllabus or paper when changing course coverage.
4. Do not present predicted questions as leaked or guaranteed.
5. Run `npm run lint` and `npm run build`.
6. Test desktop and a mobile viewport.

## Solution requirements

- Show every algebraic step needed by a beginner.
- Allocate evaluation marks that exactly equal the question marks.
- Distinguish official questions, OCR transcriptions and independently authored mock questions.
- Verify numerical answers before submission.
- Include a simple intuition sentence without replacing the formal method.

## Pull request description

Explain the problem, the change, the source used for mathematical verification and the checks performed.
